## Detalle de la Funcionalidad: Sistema de Reservas de Cápsulas

El sistema de reservas es uno de los elementos más importantes del proyecto **CapsuleCorp**.  
Su objetivo es permitir que los usuarios puedan *buscar*, *seleccionar* y *reservar* una cápsula de descanso de forma rápida y totalmente digital.

Este módulo se conecta con el área de gestión interna para que el personal del negocio pueda controlar disponibilidad, ocupación y tiempos de uso.

### Flujo de uso de la funcionalidad

1. El usuario accede a la página principal y selecciona la sección **Reservar cápsula**.
2. Introduce fecha, hora y duración del uso.
3. El sistema muestra únicamente las cápsulas disponibles.
4. El usuario elige una cápsula y confirma la reserva.
5. Se genera un código QR con los datos de acceso.
6. La reserva queda guardada automáticamente en el panel de administración.

### Tipos de usuarios implicados

- **Cliente**: puede realizar reservas, ver su historial y modificar o cancelar una reserva.
- **Administrador**: controla disponibilidad, gestiona horarios y revisa estadísticas.
- *Técnico de mantenimiento*: revisa cápsulas y marca estados como “en mantenimiento”.

### Características clave

- Gestión automática de disponibilidad.
- Confirmación inmediata de reservas.
- Integración con códigos QR.
- Interfaz sencilla para reducir el tiempo de reserva.
- Control de estados de cada cápsula (Libre, Ocupada, Limpieza, Mantenimiento).

> "Una reserva rápida y sin complicaciones mejora la experiencia del cliente y aumenta la rotación de cápsulas."
